MOTIS 2 API

Transitous provides the MOTIS 2 API. It is the first stable and versioned MOTIS API, and is incompatible with previous development versions of the API.

Our API endpoint is located at https://api.transitous.org/api/v1/.

Legacy API (MOTIS 0.x)

For a few more weeks, the legacy API is available at https://routing.spline.de/api/ for compatibility with apps that have not been updated yet.

It will still be provided until at least 2025-02-23.

The data on the legacy server is no longer kept up to date.
